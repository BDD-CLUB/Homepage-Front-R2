## 💡 시작 가이드

- nvm 설정
    
    ```
    nvm install
    nvm use
    ```
    
- root path에 `.env` 파일 추가
    
    ```
    REACT_APP_API_URL={{ apiUrl }}
    HTTPS=true
    ```
    
- 필요한 패키지 설치 및 실행
    
    ```
    npm install
    npm start
    ```
